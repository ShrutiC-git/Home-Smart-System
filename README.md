# Smart-Home-System

#### A smart home automation system built using a 3-way communication system from/to our sensors to/from our computer via a Raspberry-Pi. Communication happens via the standard MQTT protocol.
#### Pub/Sub topics for different functions around the house which have been automated.

Sensors include:
- Temperature Detector
- Moisture-Sensitivity Detector
- Smoke Detector
- Motion Detector

Designed using a breadboard </br>
Uses Arduino IDE for development and Mosquito for looking at payloads
